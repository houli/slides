Arrays and Lists

Arrays let us create lists of data

We use indexing to access the individual parts of the list


names = ["eoin", "emmet", "laura", "john"]
puts names[0]
# eoin
puts names[2]
# laura
puts names[1..3]
# emmet, laura, john

Using each

each lets us run the same code for each item in our array

The code inside do/end is called a block.


nums = [12, 30, 3, 5, 8, 1, 4]
nums.each do |number|
  if number % 2 == 0
    puts "#{number} is even."
  else
    puts "#{number} is odd."
  end
end

Using each_with_index


questions = ["Capital of Ireland?", "Lbs in a stone?", "3.0 divided by 4.0?"]
answers = ["Dublin", 14, 0.75]
questions.each_with_index do |question, index|
  puts "#{index}: #{question} ANSWER: #{answers[index]}"
end

Splitting up input into arrays

We'll use split to make an array.

Example: Average of a list of entered numbers


puts "Enter a list of numbers separated by commas"
input = gets.chomp
numbers = input.split(",")

total = 0.0
numbers.each do |number|
  total += number.to_f
end
puts "Average is #{total / numbers.length}"

Hashes

Used to group associated data


first_q = { question: "Capital of Ireland?", answer: "Dublin" }
puts first_q[:question]
# Capital of Ireland?
puts first_q[:answer]
# Dublin

Redoing our quiz question example


quiz = [{ question: "Capital of Ireland?", answer: "Dublin" },
        { question: "Lbs in a stone?", answer: 14 },
        { question: "3.0 divided by 4.0?", answer: 0.75 }]

quiz.each_with_index do |q, index|
  puts "#{index}: #{q[:question]} ANSWER: #{q[:answer]}"
end
