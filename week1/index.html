Reading in text

Actual useful input!

We'll use the gets function to "get" a string from the user


puts "What is your name?"
name = gets
puts "Hello #{name}. How old are you?"
age = gets
puts "Ah, so you're #{age}."

Let's fix that

chomp removes extra newlines


puts "What is your name?"
name = gets.chomp
puts "Hello #{name}. How old are you?"
age = gets.chomp
puts "Ah, so you're #{age}."

Much better!
